# DevConf 2026 🚀

A responsive landing page for **DevConf 2026**, a fictional developer conference — built with plain HTML and CSS.

## 🔗 Live Preview

👉 [nafismahamudshahin.github.io/DevConf-2026](https://nafismahamudshahin.github.io/DevConf-2026/)

Or run it locally — open `index.html` in your browser, no build step needed.

## ✨ Features

- **Navbar** with logo and nav links (Speakers, Schedule, Tracks, Venue, Blog)
- **Hero section** with headline, description, and a "Register Now" CTA
- **Meet the Speakers** — card grid showcasing speaker photo, track tag, name, and role
- **Pricing section** — Standard / Pro / Team ticket cards with feature lists
- **Contact & FAQ section** — contact form on the left, expandable FAQ accordion (`<details>`) on the right
- **Footer** with logo and social icons (Facebook, LinkedIn, YouTube, Twitter, Instagram)
- Fully responsive layout using Flexbox and Grid

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- [Font Awesome](https://fontawesome.com/) for icons
- [Google Fonts](https://fonts.google.com/) — Inter & Roboto
